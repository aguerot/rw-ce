# How it works

Using [uce](https://github.com/WebReflection/uce) and [sam-pattern](https://www.npmjs.com/package/sam-pattern) to demonstrate how web components work in real world application

# Getting started

```bash
yarn install

yarn build
```

# Tests

Build `tape.js`
```
yarn build-tape
```

Run tests

```
yarn test
```
